# نظام نقاط البيع (POS System)

نظام نقاط بيع بسيط مبني باستخدام Flask و Bootstrap.

## المميزات

- إدارة المنتجات (إضافة، عرض)
- نقطة بيع سهلة الاستخدام
- تتبع المبيعات وعرض التقارير
- واجهة مستخدم متجاوبة باستخدام Bootstrap
- دعم اللغة العربية

## متطلبات التشغيل

- Python 3.7+
- Flask
- Flask-SQLAlchemy
- Bootstrap 5
